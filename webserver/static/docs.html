Introduction

Welcome to the GoServer documentation. GoServer is a high-performance, production-ready HTTP server framework designed for modern web applications. It combines the raw speed of Go with developer-friendly abstractions.

Note

This documentation assumes basic familiarity with the Go programming language and HTTP concepts.

Why GoServer?

GoServer was built to solve common pain points in Go web development. While the standard library is excellent, it requires significant boilerplate for production features like graceful shutdown, structured logging, and middleware chaining.

Basic Usage

Here's a simple example of how to start a server:

package main

import (
    "github.com/goserver/core"
)

func main() {
    app := core.New()

    app.GET("/", func(c *core.Context) {
        c.JSON("Hello World")
    })

    app.Start(":8080")
}

Features

GoServer comes battery-included with everything you need:

  • Zero-allocation router
  • Middleware support
  • Group routing
  • JSON validation
  • Error management
